<h1> Enviorment Setup </h1>
<h2>WSL Install</h2>

It's really not that difficult

1. Open Windows Command Prompt.
2. Enter ```wsl --install --no-distribution```.
3. Restart your system.
4. Re-open Windows Command Prompt.
5. Enter ```wsl --install```.
6. Create your unix-login.
